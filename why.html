The following document is an example of the limitations of using inline styles (styles that are defined right inside the html tags.) Suppose you wanted to change all of the green fish words to a different color; you would have a lot of typing to do! Notice, however, that all of the words that have fish are using a span tag to define the style, and all of them are defined exactly the same way.

When you have a lot of style definitions that are going to be exactly the same, this is the perfect time to make use of internal style sheets, or sheets that apply globally to your entire document. Open up the document why2.html to see the same document defined a different way.
